# vue-project

### 项目说明

---

​	本项目是为了总结Vue所学知识，并将其体系化而建立。记录从零开始搭建一个Vue项目所考虑的各个方面以及碰到的各个问题。本项目使用vue-cli4.3版本初始化，使用了VueRouter和Vuex插件，单元测试使用Jest。

## Project setup

```
npm install
```
